# Street Coin Runner 152

An arcade endless runner desktop game built with **Electron**, **HTML5 Canvas**, **JavaScript**, **CSS**, and **Node.js**.

You're a street runner. Dodge cones, cars, trash cans, and roadblocks; collect coins; survive as long as you can.

## Features

- Main menu with Play / How to Play / Settings / Quit
- 4-lane endless runner with progressively faster speed
- Coin collecting, score, persistent high-score and total-coin saves
- Jumpable obstacles (cones, trash cans) vs blocking obstacles (cars, roadblocks)
- Animated canvas-drawn graphics - no external assets required
- Generated Web Audio API sound effects and background music
- Settings: music toggle, SFX toggle, difficulty (Easy / Normal / Hard)
- Pause / Resume, Game Over with restart
- Save data persisted in localStorage

## Controls

| Action | Key |
|---|---|
| Move left | `A` or `Left Arrow` |
| Move right | `D` or `Right Arrow` |
| Jump | `Space` / `W` / `Up Arrow` |
| Pause | `P` or `Esc` |

## Requirements

- Node.js 18+ (tested on Node 24)
- Windows 10 or 11 (for building the Windows installer)

## Running in development

```bash
npm install
npm start
```

This launches the Electron app directly from the source files.

## Building a downloadable Windows .exe

```bash
npm install
npm run build
```

The packaged output lands in the `dist/` folder:

- `StreetCoinRunner152-1.0.0-x64.exe` - the **NSIS installer** (creates desktop shortcut and Start Menu entry; user chooses install location)
- `StreetCoinRunner152-Portable-1.0.0.exe` - **portable** single-file build (no install required, just double-click)

Other build scripts:

```bash
npm run build:installer   # NSIS installer only
npm run build:portable    # portable .exe only
```

```
  README.md            this file
  src/

```

## Notes

- Save data is stored under Electron's localStorage for the app. Use the **Reset Save Data** button in Settings to clear it.
- All sprites are drawn with Canvas primitives - no images are required to ship the game.
- Music and SFX are synthesized live via the Web Audio API.
